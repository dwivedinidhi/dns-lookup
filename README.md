# DNS Lookup

An advanced DNS lookup CLI tool with colored table output.

## Features

* **query**: Query DNS records for a domain
* **reverse**: Reverse DNS lookup (PTR)
* **trace**: Trace DNS resolution path
* **batch**: Query multiple domains from a file concurrently
* **whois**: WHOIS lookup for a domain

## Usage

```bash
./dnslookup.py <command> -h
```
